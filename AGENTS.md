# Project Jam — Style Guide

## Investment Thesis Style (always apply)
- Output must be concise yet extremely specific.
- Every investment idea must be underwritten by a single, measurable catalyst.
- Prefer real tickers and dates over vague statements.
- When refining prompts, always reduce word count while increasing precision.

## Preferred Prompt Structure
Brainstormer must return ideas in exact markdown format with:
- Idea:
- Actionable Opportunity:
- Underwritten Catalyst:
- Supporting Arguments:

Critic must return:
- Counterarguments:
- Constructive Tweaks:
- Revised Action (if needed):

## Tone & Quality Bar
- Think like a hedge-fund analyst writing a 1-page memo for a PM.
- Zero filler. Every sentence must move the investment case forward.
- Allow creativity only through 2nd-order effects — never in the writing style.
